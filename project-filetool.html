Overview

   This project is a responsive and functinal calculator. The main goal was to solidify my understanding of javascript's event listeners and manipulating the DOM to create an interactive user experience.

Key Features
  • Perform addition, subtraction, multiplication and division.
  • Fully responsive design for desktop and mobile use.
  • Includes "All Clear (AC)" and "Delete (DL)" functionality.
  • Clean, intuitive user interface with a modern dark theme.
